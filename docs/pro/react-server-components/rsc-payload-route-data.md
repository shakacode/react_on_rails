# RSC Payloads as Route Data

This route is kept for compatibility with older docs links.

The standalone `createRscPayloadNode` route-data helper has been removed. Use `RSCRoute` as the canonical way to render Server Components inside client-router routes.

For loader-based router examples, see [Client router loaders](./inside-client-components.md#client-router-loaders). For the full setup, see [Embedding Server Components in Client Components](./inside-client-components.md).
