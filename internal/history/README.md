# Historical Archive

This directory preserves historical React on Rails documents that are no longer
maintained but are kept for reference. They are not linked from the active
documentation.

- **`NEWS.md`** — release and announcement timeline (last entry October 2020),
  superseded by [`CHANGELOG.md`](../../CHANGELOG.md).
- **`PROJECTS.md`** — point-in-time list of sites using React on Rails (April 2018).
- **`KUDOS.md`** — collected user testimonials (2018); some embedded images may
  no longer resolve.

These are frozen snapshots — relative links inside them may reference paths that
have since moved.
